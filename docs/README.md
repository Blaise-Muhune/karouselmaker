# Docs

Project and environment documentation.

## Backend structure

- `lib/server/` — server-only utilities and server actions
- `lib/db/` — database access and queries (Supabase)
- `lib/ai/` — AI / LLM integration for carousel content
- `lib/renderer/` — carousel slide rendering and export
- `lib/supabase/` — Supabase browser and server clients

## Environment variables

See root `.env.example` and [Environment variables](env.md) for required variables.
